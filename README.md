# 🪙 Gold Price Prediction using Machine Learning

## 📌 Overview
This project aims to predict gold prices using Machine Learning techniques based on historical data. It involves data preprocessing, visualization, and building predictive models to forecast future gold prices.

The project is implemented using Python in Google Colab.

---

## 🚀 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of gold price trends
- Machine learning model training
- Price prediction
- Model evaluation

---

## 🛠️ Tech Stack
- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
